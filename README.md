## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/HamesInFlames/Koharu-Sushi.git
cd Koharu-Sushi

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build & Deploy

```bash
# Build all projects (hub + demos)
npm run build

# Start production server
npm start
```
